# IdeaValidationAI 🚀

MVP de uma plataforma inteligente para **validação e desenvolvimento de ideias de apps, sites e programas**, utilizando IA para transformar ideias iniciais em projetos estruturados.

## 📌 Sobre o projeto
O projeto nasceu da necessidade de resolver um problema comum entre estudantes, desenvolvedores e empreendedores:

> ter boas ideias, mas não saber como validá-las, estruturá-las ou decidir se vale a pena investir.

A proposta é permitir que o usuário insira uma ideia e receba uma análise inicial com apoio de IA.

---

## 💡 Problema resolvido
Muitas pessoas têm ideias para soluções digitais, mas enfrentam dificuldades como:

- falta de clareza na estruturação
- ausência de validação de mercado
- incerteza sobre público-alvo
- dificuldade em definir próximos passos

---

## ⚙️ Funcionalidades atuais (MVP)
Atualmente, o sistema permite:

- entrada de ideia pelo usuário
- análise inicial com IA
- sugestão de estrutura 5W2H
- definição de público-alvo
- próximos passos recomendados
